# Remote Control — Mobile Access

## Deskripsi
Memungkinkan kamu mengontrol sesi Claude Code lokal dari perangkat mobile, sehingga bisa memberikan instruksi kepada tim AI dari mana saja.

## Cara Mengaktifkan

### Langkah 1: Jalankan Remote Control di Terminal
```bash
# Di terminal Claude Code (VS Code atau CLI)
claude /remote-control
```

### Langkah 2: Dapatkan Link
Setelah menjalankan perintah, Claude akan memberikan URL unik yang bisa dibuka di browser mobile.

### Langkah 3: Buka di Mobile
1. Buka link di browser HP
2. Atau buka aplikasi Claude di HP dan gunakan session yang sama
3. Ketik instruksi seperti biasa

## Use Cases Mobile

### Quick Tasks dari HP:
```
"Cek ada tugas apa hari ini di Notion"
"Tulis caption Instagram untuk Rancabango, tema weekend getaway"
"Berapa CPL campaign Gwesha minggu ini?"
"Buatkan draft brief untuk campaign bulan depan"
```

### Monitoring:
```
"Status task yang sedang dikerjakan?"
"Ada output baru yang sudah selesai?"
"Ringkasan performa sosmed hari ini"
```

### Approval Flow:
```
"Tampilkan draft caption yang sudah dibuat"
"Approve content calendar bulan ini"
"Revisi headline campaign jadi lebih catchy"
```

## Tips Penggunaan Mobile
1. Gunakan perintah singkat dan jelas
2. Manfaatkan `/` commands untuk aksi cepat
3. Review output saat di mobile, revisi detail saat di desktop
4. Cocok untuk approval dan monitoring, bukan heavy editing

## Keamanan
- Link remote control bersifat sementara (per session)
- Hanya bisa diakses oleh pemegang link
- Session tetap berjalan di komputer lokal
- Jangan share link ke pihak yang tidak berkepentingan
