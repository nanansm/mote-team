# Skill: Branded Deck Creator

## Deskripsi
Membuat presentasi/proposal/offering document yang mengikuti identitas visual Mote Kreatif berdasarkan template referensi yang sudah ada.

## Kapan Digunakan
- Membuat proposal/offering untuk klien baru
- Membuat presentasi strategi kampanye
- Membuat pitch deck
- Membuat laporan bulanan dalam format presentasi

## Referensi Template
Lihat file template di `templates/presentations/` untuk referensi visual dan struktur.

## Struktur Proposal Offering Mote (Standar)

### Slide 1: Cover
- Logo Mote Kreatif (kiri atas)
- Label "PROPOSAL DOCUMENT" (kanan atas)
- Judul besar (bold, hitam)
- Info: Prepared For [Klien], Jenis Layanan, Tanggal
- Footer: "Garut, Indonesia" | "Creative Agency © 2026"
- Background: Kuning/Oranye (#E8A000)

### Slide 2: Project Objective
- Label "PROJECT OBJECTIVE"
- Statement masalah klien (bold, besar)
- Paragraf solusi yang ditawarkan Mote
- Footer: Logo Moté kecil
- Background: Krem/abu muda (#D9D5C9)

### Slide 3: Project Profile
- Label "PROJECT PROFILE"
- Tabel info: Client, Brand Name, Brand Type
- Tabel: Intention, Project Specs
- Tabel: Website, Social Media, Location
- Tabel: Duration Plan, Arranged By
- Background: Putih

### Slide 4: Target & Impact
- Label "TARGET & IMPACT"
- Sub-label: "TUJUAN PROJECT DIGITAL MARKETING"
- 3-5 tujuan dalam kotak/kolom
- Masing-masing dengan deskripsi pendek
- Background: Putih

### Slide 5: Offering Detail
- Label "OFFERING DETAIL"
- Issue Date, Nomor dokumen, Detail layanan
- Tabel layanan (2-4 kolom sesuai scope)
- Struktur Team (numbered list)
- Investment Marketing Fee (tabel harga)
- Background: Putih

### Slide 6: Closing
- "Hatur Nuhun. Saatnya berkarya untuk kemajuan bersama."
- Ilustrasi tim Mote
- Kontak: Email, Telepon, Website, Instagram
- Background: Kuning/Oranye (#E8A000)

## Instruksi untuk Claude

### Input yang Dibutuhkan:
1. Nama klien & brand
2. Jenis bisnis/industri
3. Masalah/kebutuhan klien
4. Layanan yang ditawarkan
5. Harga/fee
6. Durasi project
7. Tim yang terlibat

### Proses:
1. Baca template referensi di `templates/presentations/`
2. Sesuaikan konten dengan input klien
3. Pertahankan struktur dan tone visual Mote
4. Generate file presentasi
5. Simpan di `workspace/presentations/`

### Naming Convention:
`MOTE-[KODE_LAYANAN].[DD-MM]/[YY]`
Contoh: `MOTE.DM.26-05/25` (Digital Marketing, 26 Mei 2025)

## Tips Kualitas
- Gunakan bahasa yang percaya diri tapi tidak arogan
- Masalah klien ditulis bold dan besar (empati)
- Solusi Mote ditulis lebih kecil sebagai supporting text
- Selalu sertakan track record/case study yang relevan
- Harga ditampilkan jelas dalam tabel
